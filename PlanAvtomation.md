##### 1. Отправка заполненной формы кнопкой "Записаться".

        Шаги:
            1. Заполнить поле "Имя" именем на кириллице

##### Инструменты

1.Intel Idea Ultimate -  написание кода
2. JUnit5 - платформа для написания автотество и их запуск.
3. Postman - проверка запросов
4. Gradle - универсальная система управления проектами.
5. Selenide -фреймворк для автоматизированного тестирования вебприложений.
6. Docker - система контейнеризации, которая поможет работать с базой данных
7. Docker compose - удобрый инструмент, позволяющий запускать мультиконтейнерные приложения.
8. Allure -  фреймворк, предназначенный для создания визуально наглядной картины выполнения тестов.
9. DevTools  - Консоль разработчика для поиска селекторов
